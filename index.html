What is Steganography?

  • Hidden in Plain Sight: The art of hiding information inside other information.
  • Invisible Secrecy: Unlike encryption which scrambles data, steganography conceals the existence of the message itself.
  • Digital Media: Commonly uses images, audio, or video files as "carriers".

LSB Steganography

We modify the Least Significant Bit of the pixel's color value.

RGB: (214, 171, 204)
Red Bin: 11010110

Hiding bit '1':
Red Bin: 11010111

New RGB: (215, 171, 204)
(Difference is invisible to human eye)

Security Features

Input Validation

Text sanitization (max 10k chars) and Key length limits prevent buffer overflows and injection attacks.

XSS Protection

Frontend uses `textContent` instead of `innerHTML` to prevent raw HTML injection during message display.

Key Verification

Embedded verification markers ensure that incorrect keys are detected immediately before attempting decryption.

Image Constraints

Strict file type validation (PNG/JPG) and automatic resizing to prevent DoS attacks via massive files.
